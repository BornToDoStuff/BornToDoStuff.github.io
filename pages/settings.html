Settings

These settings use cookies in your browser to change aspects about the site.


Table Mode

Switch items between grid and table view. Table view hides images and text but fits more things in a single screen.

Detailed View

View item costs, tags, and other details that are otherwise hidden for easy viewing when visiting item pages.

Advanced Search

Enable a more advanced search that is more finicky but allows for more complex filtering and sorting.
